# Mirai-NLP-Notebook
Mirai-NLP训练笔记本

## 使用方法

- 收集并导出对应格式数据
- 在[笔记本列表](#笔记本列表)找到对应模型的Colab笔记本并进入
- 登录 Google 账号并确保 Google Drive 可用
- 首先点击右上角的**连接**，等待分配完成和初始化
- 在上方菜单栏中选择**代码执行程序**，在倒数第三项找到**更改运行时类型**，在**硬件加速器**中选择 `GPU`，等待分配完成
- 按照笔记本提示进行初始化、训练和生成
